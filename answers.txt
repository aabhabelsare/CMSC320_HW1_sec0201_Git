
Q1: Why might we consider using Git?

Answer: We would consider using Git because it is a helpful tool when it comes to collaborating on code with other people. Using Git provides a way to sync changes from several people together, which makes communication on a project much simpler.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to the healthcare industry. For example, data science can be used to analyze patterns in patient data and maybe predict diseases or malfunctions in the patient earlier, which can improve treatments and improve patient care by being more proactive.
